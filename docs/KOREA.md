# 한국 크리에이터 가이드

한국에서 모르는 사람에게 소액 팁을 받을 수 있는 현실적인 수단과 설정 방법입니다. (2026-06 기준 조사 — 한국 결제 서비스는 자주 바뀌니 링크가 살아있는지 주기적으로 확인하세요.)

## 1. 카카오페이 송금코드 — 기본 추천

- **설정**: 카카오톡 > 더보기 > 코드스캔 > 하단 '송금코드' 탭 > 공유 → `qr.kakaopay.com/...` 링크를 그대로 `links`에 붙여넣기. 약 1분, 사업자등록 불필요, 수수료 0원.
- **알아둘 것**: 송금자에게 본인 실명이 표시됩니다. 받은 돈은 카카오페이머니로 들어옵니다(기본 보유한도 200만원, 증권계좌 연결 시 해제). VibeTip은 모바일에서 송금 링크를 바로 열고, 데스크톱에서는 휴대폰으로 스캔할 QR을 표시합니다.
- developers.kakaopay.com의 송금 API는 사업자 제휴 전용이니 개인은 신청할 필요 없습니다.

## 2. 크티 (CTEE) — 웹 결제가 필요하면

- **설정**: [ctee.kr](https://ctee.kr)에서 SNS 계정으로 가입(약 3분, 사업자등록·심사 불필요) → `ctee.kr/place/{아이디}` 링크 사용.
- **수수료**: 플랫폼 0% + PG 약 3.2%. 결제 즉시 정산. 해외 팬용 영문 페이지 자동 제공.
- 데스크톱·모바일 모두 일반 웹 결제로 동작하는 게 강점입니다.

## 3. 투네이션 — 세금 처리가 가장 간단

- **설정**: [toon.at](https://toon.at) 가입 → 본인인증 정산 계약 → `toon.at/donate/{아이디}`.
- **수수료**: 결제수단별 (카드 약 6.3%) — 단, **원천징수 3.3%가 포함**되어 후원 규모가 커져도 세금 신고 부담이 가장 적습니다.
- **단점**: 후원자가 로그인 + 투네캐시 충전을 해야 해서 처음 방문자에게 문턱이 있습니다.

## 4. GitHub Sponsors — 해외·개발자 후원용

- **설정**: 한국 은행 계좌 + W-8BEN 양식으로 등록(약 1일 승인). 개인 스폰서십 플랫폼 수수료 0%, USD→KRW 환전 약 2%.
- 한국 은행으로 실제 정산되는 거의 유일한 글로벌 레일입니다. 카카오페이(국내용)와 함께 거는 조합을 추천합니다.

## 5. 토스 송금 딥링크 — 이해하고 쓰는 사람만

토스아이디(toss.me)는 **2024-08-01 종료**됐습니다. 남아있는 방법은 비공식 딥링크뿐입니다:

```
supertoss://send?amount=5000&bank=토스뱅크&accountNo=계좌번호&origin=qr
```

- **경고**: URL 자체에 실계좌번호가 들어가고, 송금자에게 실명이 표시됩니다. 비공식 기능이라 토스가 예고 없이 막을 수 있습니다.
- 데스크톱에서는 동작하지 않습니다(모바일 + 토스 앱 필요).

## 글로벌 플랫폼 주의사항

| 플랫폼 | 한국 크리에이터 |
|---|---|
| Buy Me a Coffee | ❌ 2024-11-01 Payoneer 정산 중단 이후 한국 수령 불가 |
| Stripe | ❌ 한국에서 가맹 계정 개설 자체가 불가 |
| Ko-fi / PayPal | ⚠️ 해외 팬 전용 — 한국 팬은 PayPal 국내 결제가 막혀 있고, 원화 출금에는 사업자등록번호 필요 |

사업자등록 없는 개인 카드결제 서비스(페이앱 라이트)는 2025-03-10 금융위 결정으로 종료되어 더 이상 존재하지 않습니다.

## 세금 안내 (법률·세무 자문이 아닙니다)

후원금은 명목과 무관하게 과세 대상입니다(국세청 1인미디어 창작자 세무안내 기준):

- **계속·반복적으로** 받는 후원은 사업소득 — 사업자등록(통상 면세사업자, 업종코드 940306)과 다음 해 5월 종합소득세 신고 대상입니다.
- **일시적** 후원은 기타소득(연간 기타소득금액 300만원 이하면 분리과세 선택 가능), 대가 없는 순수 증여로 보는 경우 증여세 대상이 될 수 있습니다.
- 카카오페이·토스 등 **P2P 송금은 원천징수가 없으므로** 전액 자진 신고 책임이 본인에게 있습니다. 2026년부터 연 500만원 이상 현금성 직접수입은 현금매출명세서 제출 의무(누락 시 가산세)가 적용됩니다.
- 후원이 정기화되면 원천징수를 대신 처리해주는 플랫폼(투네이션·크티)을 P2P 직접 송금보다 우선 고려하고, 세무사 상담을 권합니다.

문구는 반드시 **"개인 창작자 후원/팁"** 으로 표현하세요. "공익 기부/모금"으로 비치면 기부금품법상 등록 의무(연 1천만원 이상 모집 시)가 발생할 수 있습니다.

## VibeTip의 책임 범위

VibeTip은 자금을 보관·처리·중개하지 않으며 링크만 렌더링합니다. 정산·세금·법적 의무는 전적으로 크리에이터와 선택한 결제 레일에 있습니다.
