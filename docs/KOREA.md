# 카카오페이 송금코드 가이드

VibeTip은 현재 카카오페이 송금코드만 지원합니다.

## 송금 링크 준비

1. 카카오톡 또는 카카오페이에서 본인의 송금코드를 엽니다.
2. 송금코드를 공유해 `https://qr.kakaopay.com/...` 형태의 URL을 복사합니다.
3. 복사한 URL을 VibeTip의 `links`에 넣습니다.

```ts
import { init } from 'vibetip'

init({
  name: '홍길동',
  links: ['https://qr.kakaopay.com/your-code'],
})
```

## 기기별 동작

- 모바일: 카카오페이 송금 링크를 새 창으로 엽니다.
- PC: 페이지를 이동하지 않고 위젯 안에 송금 QR을 표시합니다.
- QR 화면에서 뒤로 가기 또는 `Escape`를 누르면 송금 버튼으로 돌아갑니다.

## 지원 범위

허용되는 호스트는 다음 두 가지입니다.

- `qr.kakaopay.com`
- `link.kakaopay.com`

다른 결제·후원 URL을 전달하면 위젯은 초기화되지 않고 오류를 반환합니다.

## 운영 전 확인

- 실제 휴대폰에서 링크가 본인의 송금 화면으로 연결되는지 확인하세요.
- PC에서 QR을 스캔했을 때 같은 송금 화면이 열리는지 확인하세요.
- 공개 페이지에 올리기 전에 송금 화면에 표시되는 수취인 정보를 확인하세요.
- 후원·팁 수입의 신고 및 세무 처리는 개인 상황에 따라 달라질 수 있으므로 필요한 경우 세무 전문가에게 확인하세요.

## VibeTip의 책임 범위

VibeTip은 자금을 보관·처리·중개하지 않습니다. 카카오페이 송금 링크와 QR을 표시하는 UI만 제공합니다.
